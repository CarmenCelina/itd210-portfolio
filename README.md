# Portfolio Website — Carmen Cruz

## About This Project
This is my capstone project for my web design class. This website is responsive and uses CSS Grid and Flexbox for layout. It showcases a local charter bus service. All contact and booking information is available for customers to easily find.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Responsive Design)
- JavaScript
- Milestone 3
- added lazy loading & perfers-reduced-motion
